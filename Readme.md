# 📘 Java Basic - Tổng hợp bài tập Java cơ bản

Kho lưu trữ này chứa các project bài tập thực hành ngôn ngữ lập trình Java, từ các phép toán cơ bản đến lập trình hướng đối tượng (OOP) và xử lý luồng dữ liệu (Stream IO).

## 🚀 Hướng dẫn cách chạy các Project

Để chạy các bài tập này trên máy tính, bạn thực hiện theo các bước sau:

1. **Yêu cầu hệ thống:** Máy tính cần cài đặt **JDK (Java Development Kit)** phiên bản 8 trở lên.
2. **Cài đặt IDE:** Khuyến khích sử dụng **Eclipse**, **IntelliJ IDEA** hoặc **NetBeans**.
3. **Các bước thực hiện:**
   - Clone kho mã nguồn về máy: 
     ```bash
     git clone [https://github.com/trunglmtech/Bai1-Java-Basic.git](https://github.com/trunglmtech/Bai1-Java-Basic.git)
     ```
   - Mở IDE và chọn **Import Project** (chọn thư mục của project cụ thể bạn muốn chạy).
   - Tìm tệp tin `.java` có chứa hàm `public static void main(String[] args)`.
   - Nhấn chuột phải và chọn **Run As > Java Application**.

### 📝 Lưu ý về Ảnh Code Demo
Để các hình ảnh demo hiển thị chính xác trong bảng trên, bạn hãy:
1. Chụp ảnh màn hình code hoặc kết quả chạy chương trình.
2. Lưu ảnh đó vào thư mục của project tương ứng với tên là `demo.png`.
   * Ví dụ: Ảnh của bài `GiaiPTBac1` sẽ nằm tại đường dẫn `GiaiPTBac1/demo.png`.
3. Sau khi `git push` lên GitHub, ảnh sẽ tự động xuất hiện trong file Readme.
